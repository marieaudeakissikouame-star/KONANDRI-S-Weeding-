# Configuration Google Drive & Google Sheets\n\n## Variables à configurer\n\nEditez `js/app.js` et remplacez :\n\n### Google Drive\n```javascript\nconst GOOGLE_DRIVE_FOLDER_ID = 'YOUR_FOLDER_ID_HERE';      // ID du dossier Drive partagé\nconst GOOGLE_DRIVE_API_KEY = 'YOUR_API_KEY_HERE';          // Clé API Google Cloud\n```\n\n**Comment obtenir :**\n1. Dossier Drive : copiez l'ID de l'URL `drive.google.com/drive/folders/ID`\n2. API Key : [console.cloud.google.com](https://console.cloud.google.com) → Google Drive API → Clé API publique\n\n### Google Sheets\n```javascript\nconst SHEET_ID = 'YOUR_SHEET_ID_HERE';        // ID du Google Sheet\nconst TAB_RSVP = 'Réponses au formulaire 1';  // Nom exact de l'onglet 1\nconst TAB_TABLES = 'Table des invités';       // Nom exact de l'onglet 2\n```\n\n**Comment obtenir :**\n- Copiez l'ID depuis l'URL `docs.google.com/spreadsheets/d/ID`\n- Vérifiez les noms d'onglets (sensibles à la casse)\n\n---\n\n## Onglet 1 : Réponses au formulaire 1\n\n| Timestamp | Prénom Nom | Accompagnant? |\n|-----------|-----------|---------------|\n| 10:00 AM  | Marie Aude | Oui           |\n| 10:05 AM  | Yoane K   | Non           |\n\n- **Colonne A** : Date/heure\n- **Colonne B** : Nom complet\n- **Colonne C** : Oui/Non pour accompagnant\n\n---\n\n## Onglet 2 : Table des invités\n\n| Nom        | Table | Place |\n|-----------|-------|-------|\n| Marie Aude | 1     | 1     |\n| Yoane K   | 1     | 2     |\n\n- **Colonne A** : Nom complet (doit matcher Onglet 1)\n- **Colonne B** : Numéro de table\n- **Colonne C** : Numéro de place\n\n---\n\n## Partage\n\n✅ Google Drive : **Public** (lecture)\n✅ Google Sheets : **Public** (lecture)\n\n---\n\n## Test\n\n1. Modifiez Google Sheets\n2. Actualisez le site\n3. Recherchez un nom dans \"Trouver ma place\"\n4. Les données devraient s'afficher ✓\n\nVoir `SETUP.md` pour le guide complet.\n"
