# 🚀 Setup — Google Drive + Google Sheets\n\n## Google Drive (Galerie collaborative)\n\n### 1. Créer un dossier Google Drive partagé\n\n1. Ouvrez [Google Drive](https://drive.google.com)\n2. Cliquez **Nouveau** → **Dossier**\n3. Nommez-le : `Mariage Yoane & Marie-Aude - Photos`\n4. **Partagez** avec \"Accès public\" (pour lire)\n5. Copiez l'ID du dossier depuis l'URL :\n   ```\n   https://drive.google.com/drive/folders/YOUR_FOLDER_ID\n                                        ↑ Copiez ceci\n   ```\n\n### 2. Activer Google Drive API\n\n1. Allez sur [Google Cloud Console](https://console.cloud.google.com)\n2. Créez un nouveau projet ou utilisez un existant\n3. Activez **Google Drive API** :\n   - Recherchez \"Google Drive API\"\n   - Cliquez \"Activer\"\n4. Créez une **clé API** publique :\n   - Allez dans \"Identifiants\"\n   - Cliquez \"Créer identifiant\" → \"Clé API\"\n   - Copiez la clé générée\n5. **Limitez la clé** à Google Drive API uniquement\n\n### 3. Configurer dans le site\n\nEditez `js/app.js` et remplacez :\n\n```javascript\nconst GOOGLE_DRIVE_FOLDER_ID = 'YOUR_DRIVE_FOLDER_ID_HERE';      // Étape 1\nconst GOOGLE_DRIVE_API_KEY = 'YOUR_API_KEY_HERE';          // Étape 2\n```\n\n### 4. Tester\n\n- Téléchargez des photos dans le dossier Drive\n- Actualisez le site\n- Les photos apparaîtront dans la galerie ✓\n\n---\n\n## Google Sheets (Invités + Placements)\n\n### 1. Créer le Google Sheet\n\n1. Ouvrez [Google Sheets](https://sheets.google.com)\n2. Créez un nouveau sheet\n3. **Onglet 1** - Renommez : `Réponses au formulaire 1`\n   \n   Colonnes :\n   ```\n   Timestamp | Prénom Nom      | Accompagnant?\n   --------- | --------------- | ---------------\n   10:00 AM  | Marie Aude      | Oui\n   10:05 AM  | Yoane Kouakou   | Non\n   ```\n   - **A** : Timestamp (date/heure)\n   - **B** : Prénom + Nom complet\n   - **C** : Oui/Non\n\n4. **Onglet 2** - Renommez : `Table des invités`\n   \n   Colonnes :\n   ```\n   Nom          | Table | Place\n   ------------ | ----- | ------\n   Marie Aude   | 1     | 1\n   Yoane K      | 1     | 2\n   ```\n   - **A** : Nom complet (doit matcher Onglet 1)\n   - **B** : Numéro de table\n   - **C** : Numéro de place\n\n### 2. Partager en lecture\n\n1. Cliquez **Partager**\n2. Changez à \"Accès public\" → **Lecteur**\n3. Copiez l'ID du sheet :\n   ```\n   https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID\n                                        ↑ Copiez ceci\n   ```\n\n### 3. Configurer dans le site\n\nEditez `js/app.js` et remplacez :\n\n```javascript\nconst SHEET_ID = 'YOUR_SHEET_ID_HERE';\nconst TAB_RSVP = 'Réponses au formulaire 1';  // Nom exact de l'onglet\nconst TAB_TABLES = 'Table des invités';       // Nom exact de l'onglet\n```\n\n### 4. Tester\n\n- Modifiez le Google Sheet\n- Actualisez le site\n- Les données se synchronisent automatiquement ✓\n\n---\n\n## 🔐 Sécurité\n\n⚠️ **IMPORTANT** :\n- La clé API Google Drive est publique (mais limitée à Drive)\n- Le Sheet doit être en \"Accès public\" pour la lecture\n- Pas de données sensibles ne doivent être exposées\n- Les invités ne peuvent que lire (pas modifier)\n- La clé API affichée dans le code ne pose pas de risque majeur (lecture seule)\n\n---\n\n## 📋 Checklist finale\n\n- [ ] Google Drive dossier créé et partagé public\n- [ ] Google Drive API activée (Google Cloud Console)\n- [ ] Clé API créée et limitée à Google Drive\n- [ ] `GOOGLE_DRIVE_FOLDER_ID` copié dans `js/app.js`\n- [ ] `GOOGLE_DRIVE_API_KEY` copié dans `js/app.js`\n- [ ] Google Sheet créé avec 2 onglets (noms corrects)\n- [ ] Google Sheet partagé en public (lecteur)\n- [ ] `SHEET_ID` copié dans `js/app.js`\n- [ ] Noms d'onglets vérifiés (sensibles à la casse)\n- [ ] Site testé localement\n- [ ] Site déployé sur GitHub Pages ✓\n\n---\n\n## 🧪 Test rapide\n\n### Test Drive\n```bash\n# 1.
